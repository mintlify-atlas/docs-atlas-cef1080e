Anomalib is a deep learning library that collects state-of-the-art anomaly detection algorithms for benchmarking on public and private datasets. It provides ready-to-use model implementations, standardized training and evaluation pipelines, and tools for developing custom anomaly detection solutions — all focused on visual anomaly detection in images and video.

## Get started

<Steps>
  <Step title="Install Anomalib">
    Install from PyPI with your hardware backend of choice:

    ```bash
    pip install anomalib          # standard install
    pip install "anomalib[cu126]" # CUDA 12.6
    pip install "anomalib[cpu]"   # CPU only
    ```
  </Step>
  <Step title="Train a model">
    Use the Python API to train PatchCore on the MVTec AD dataset:

    ```python
    from anomalib.data import MVTecAD
    from anomalib.models import Patchcore
    from anomalib.engine import Engine

    datamodule = MVTecAD()
    model = Patchcore()
    engine = Engine()

    engine.fit(datamodule=datamodule, model=model)
    ```
  </Step>
  <Step title="Run inference">
    Generate predictions from a trained checkpoint:

    ```python
    predictions = engine.predict(
        datamodule=datamodule,
        model=model,
        ckpt_path="results/Patchcore/MVTecAD/latest/weights/lightning/model.ckpt",
    )
    ```
  </Step>
  <Step title="Export for production">
    Export to OpenVINO for optimized edge inference:

    ```python
    from anomalib.deploy import ExportType

    engine.export(
        model=model,
        export_type=ExportType.OPENVINO,
        ckpt_path="path/to/model.ckpt",
    )
    ```
  </Step>
</Steps>

## Key features

- **20+ algorithms** — PatchCore, PaDiM, EfficientAD, DRAEM, WinCLIP, and many more
- **Dual interface** — Python API and CLI for every workflow
- **Lightning-based** — Built on PyTorch Lightning to reduce boilerplate
- **OpenVINO export** — Accelerated inference on Intel hardware out of the box
- **Multi-modal support** — Images, video, and depth data
- **Learning paradigms** — One-class, zero-shot, and few-shot learning
- **Experiment tracking** — Native integrations with W&B, Comet.ml, and TensorBoard
- **Anomalib Studio** — No-code web UI for training and camera-based deployment

<Note>
  Anomalib requires Python 3.10+ and PyTorch 2.6+. See the [installation guide](/installation) for full requirements.
</Note>
